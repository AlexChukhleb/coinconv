ТЗ:
Необходимо создать утилиту командной строки, которая конвертирует одну валюту в другую используя
https://coinmarketcap.com/api/v1/#section/Introduction как источник данных.

Пример:
$> ./coinconv 123.45 USD BTC

Программа должна вывести результат в консоль.
Исходный код плюс все зависимости должны быть размещены в отдельном репозитории на github.com или подобном сервисе.

В разработке можно использовать тестовый ключ для sandbox окружения coinmarketcap.
+ Чистая архитектура (очень желательно)


Комментарии к реализации:

Метод из предложенной документации (https://coinmarketcap.com/api/documentation/v1/#operation/getV2ToolsPriceconversion)
не работает согласно описанию. На этом моменте следовало бы прекратить тратить время на это "небольшое тестовое задание",
но раз взялся нужно доделать.

Ведется 3 пути решения с использованием coinmarketcap.com:
1. Пытаться найти условия при котором заработает /v2/tools/price-conversion, что явно выходит за рамки задачи
2. Использовать другие методы из https://coinmarketcap.com/api/documentation/v1/#tag/cryptocurrency,
  но там нужно приобрести опыт в работе биржи, если такой опыт крайне необходим о его наличии следовало
  уточнить до выдачи тестового задания
3. В описании метода есть ссылка на https://coinmarketcap.com/converter/ по которой указано, что работа эквивалента.
  Это не продакшн решение, но для достижения результата подходит.
Т.к. задача является тестовой и не предполагается к использованию в продакшене или под нагрузкой выбран вариант 3.
По той же причине для денег использован тип float64 вместо decimal
